# **App Name**: Master PAO

## Core Features:

- PAO Data Management: Loads and manages the Person-Action-Object (PAO) table data for numbers 00-99 from an internal JSON structure, including placeholders for incomplete entries from the provided document.
- Flashcard Learning Mode: Presents digital flashcards where users can flip to reveal the Person, Action, Object, and Etymology details for each number (00-99) as per the PAO table.
- The Blitz Quiz Practice: A timed practice mode that displays random numbers and requires the user to select the correct Person, Action, or Object from multiple-choice options under pressure, reinforcing instinctual recall.
- Blitz Quiz Leveling System: Optimizes quiz logic with progressive difficulty levels: Level 1 focuses on Person (00-09), Level 2 combines Person and Action (10-19), and Level 3 tests Object associations for all numbers.
- Adaptive Review Mode: Implements an algorithm to prioritize and more frequently present numbers that the user has previously answered incorrectly, ensuring targeted practice.
- Neural Strength Progress Visualization: Visualizes the memory strength and recall progress for each number based on the frequency and accuracy of correct answers.

## Style Guidelines:

- Background Color: Deep Charcoal (#121212). This dark, muted hue establishes a 'Futuristic Heritage' and 'Laboratorium Masa Depan' ambiance, providing a stark contrast for neon elements.
- Primary Color: Acid Lime (#DEFF9A). This vibrant, almost glowing lime green (H72, S100, L80) serves as the primary interactive color for highlights, text accents, and the 'Glow' effect on numbers, creating a high-tech, energetic feel against the dark background.
- Accent Color: Golden Yellow (#E0B651). An analogous accent (H42, S70, L60) that provides a complementary splash of color for secondary highlights or subtle distinctions, balancing the boldness of Acid Lime without clashing.
- Numbers and Display Font: 'Urbanist', a contemporary sans-serif font, for all prominent numbers and main headings, embodying the modern, futuristic aesthetic.
- Body Text Font: 'PT Sans', a readable and modern humanist sans-serif, for all supporting text and descriptive content, ensuring clarity and professionalism throughout the interface.
- Use minimalist, geometric icons that align with the 'Future Laboratory' theme, prioritizing clarity and subtle high-tech visual cues.
- Embrace a clean, spacious layout with card-based UI elements, emphasizing focus on the content and minimizing distractions, befitting a training module.
- Subtle, smooth transition animations using Framer Motion for card flips, quiz progression, and UI element interactions, enhancing the futuristic and polished user experience. Integrate a distinct 'Denting Logam' (metallic clang) sound effect for correct quiz answers.
